The internet is a great place to start learning! There are many websites offering free online courses including:

Code Academy

Code Academy offers online lessons in 12 different programming languages including Python, Java, JavaScript, Ruby, SQL and C++ as well as markup languages HTML and CSS. Each class ends with a task aimed to put your coding skills to test. You can find Code Academy here.

Edx

EdX is like the Amazon of STEM courses as it hosts courses from many of the highest ranking universities in the world.Want to learn Data Science through Harvard? Blockchain from Berkley? Maybe Human Neuroanatomy from Michican, they're all on EdX. You can access EdX here.

MITOCW

With videos of lectures on algorithms, linear algebra, calculus and more, the Massachusetts Institute Of Technology has developed Open Courseware so that anyone, anywhere can study. There are also tasks set to check you have understood the material.So you can learn the same as a for free, but without the college credit. You can find their website here.
